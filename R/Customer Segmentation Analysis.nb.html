Introduction

In this project, we will perform customer segmentation analysis using machine learning and K-means clustering algorithm. Customer segmentation analysis helps businesses identify potential customer groups, allowing business to tailor marketing and business strategies to each specific group.

Determining K-means Clusters

In the K-means cluster algorithm, the machine learning algorithm will create centroids around a specified number of clusters(k) based on shortest Euclidean Distance. We will explore three different methods of determining the most suitable number of clusters (k).

Elbow Method

set.seed(123)
intra_square_sum <- function(k) {
  kmeans(customer_data[3:5],k,iter.max=100,nstart=100,algorithm="Lloyd")$tot.withinss
}
k.values <- 1:10

iss_values <- map_dbl(k.values, intra_square_sum)

plot(k.values, iss_values,
     type='b', pch=19, frame=FALSE,
     xlab="Number of clusters K",
     ylab="Total intra-clusters sum of squares")

We look at the elbow plot to see where the total intra-clusters sum of square starts to diminish in its decreasing velocity. For the plot above, that would be about K = 4, or the elbow.

Silhouette Method

The silhouette method calculates the mean silhouette observations for different k values. The highest average silhouette value signifies the best k value.

Gap Statistic method

The gap statistic method compares variation in the clusters of the actual data set vs a generated random data set (which is generated via Monte Carlo simulations). This ensures ML is not just finding patterns by chance.

set.seed(125)
stat_gap <- clusGap(customer_data[,3:5], FUN = kmeans, nstart = 25,
            K.max = 10, B = 50)
Clustering k = 1,2,..., K.max (= 10): .. done
Bootstrapping, b = 1,2,..., B (= 50)  [one "." per sample]:
.................................................. 50 
fviz_gap_stat(stat_gap)

So, it looks like 6 is the optimal cluster number.

Conclusion

In this project, we performed unsupervised machine learning via k-means clustering to segment customers for potential business and market strategies.

We learned important characteristics and information of each customer group and potential relationship between their age, annual income, and spending score. We may also consider converting categorical data ie gender into numeric representation ie one-hot encoding, so that we may perform k-clustering on it as well. This may reveal potential for targeted marketing towards each gender if significant differences were found.
